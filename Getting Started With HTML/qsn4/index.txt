TAGS:

HTML tags are like keywords which defines that how web browser will format and display the conten.
 HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags.
 All HTML tags must enclosed within < > these brackets.
 Every tag in HTML perform different tasks.
 Syntax of tag:
          <tag> content </tag>

          ELEMENTS:
          An HTML element is a complete unit that includes an opening tag, content, and a closing tag.
          Example:
          <p>This is a Paragraph</p>
          In the above example <p> is an opening tag and </p> is closing tag .This in complete forms an HTML Element.
